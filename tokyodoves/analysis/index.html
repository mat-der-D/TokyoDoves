Module tokyodoves::analysis

source ·
Expand description

Convenient tools to analyze the game.

The following functions are main contents:

  • evaluate_board
    It calculates a possible range of BoardValue of specified Board.
  • compare_board_value
    It compares the value of specified Board to a given BoardValue.
  • create_checkmate_tree and create_checkmate_tree_with_value
    It creates an BoardValueTree that describes routes to ends of the game.
  • find_best_actions
    It collects the best Actions by BoardValue.

The BoardValue struct plays important roles in the all above functions. See their documentations for more.
