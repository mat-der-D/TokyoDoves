Convenient tools for playing games

Main contents:

  • Game
    A struct that provides methods to play the game.

  • Agent
    A trait to define an agent playing the game. The following agents are included:

    • RandomAgent
      An agent who chooses the next action at random.
    • AnalystAgent
      An agent who chooses the next action based on some analyses.
    • ConsoleAgent
      An agent who asks the next action to the console. It provides CLI with humans.

    You can define another agent in you code by implementing the Agent trait if you want.

  • Arena
    A struct of an arena, where two Agents play against.
